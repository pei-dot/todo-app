# とぅどぅ

シンプルで使いやすいタスク管理Webアプリです。

## 機能一覧

### タスク管理
- タスクの追加・完了・削除
- 完了したタスクは自動で下部へ移動
- 追加日のツールチップ表示（ホバーで確認）

### リスト管理
- タスクをグループ化するリストの作成
- リストの展開・折りたたみ
- リスト内へのタスク追加

### 期限設定
- カスタムカレンダーでタスクに期限を設定
- 時間指定も可能
- 期限の状態を色分けバッジで表示（将来・もうすぐ・今日・期限切れ）

### ドラッグ＆ドロップ
- タスク・リストの並び替え
- リスト間をまたいだタスクの移動
- ピン済みアイテムはピン済みグループ内でのみ移動可能

### ピン機能
- タスク・リストをピンで上部に固定
- ピン済みアイテムは常にピンなしアイテムより上に表示

### まとめて選択
- 複数のタスク・リストを一括選択
- 選択したアイテムをまとめて削除

### 履歴
- タスク・リストの追加・完了・削除・移動・期限設定を記録
- 履歴の一括クリアに対応

### データ保存
- localStorage によるデータ永続化
- ページリロード後もデータを保持

## 使用技術

| 技術 | 用途 |
|---|---|
| HTML | 構造 |
| CSS | スタイリング |
| JavaScript| ロジック・インタラクション |

ライブラリ・フレームワーク不使用。

## ファイル構成

```
├── index.html   # HTML構造
├── todo.css     # スタイル
└── todo.js      # アプリケーションロジック
```

## ローカルでの実行

```bash
git clone https://github.com/pei-dot/todo-app.git
cd todo-app
```

`index.html` をブラウザで開くだけで動作します（サーバー不要）。
